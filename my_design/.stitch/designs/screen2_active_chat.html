React Hooks 核心概念

React Hooks 是 React 16.8 引入的一项重大革新，它允许你在不编写 class 的情况下使用 state 以及其他的 React 特性。Hooks 的出现主要是为了解决组件间逻辑复用困难、复杂组件难以理解等问题。

1. useState: 状态管理

这是最基础的 Hook，用于在函数组件中添加本地状态。

const [count, setCount] = useState(0); // 使用场景：处理表单输入、切换开关状态、计数器等

能给我一个自定义Hook的例子吗？

10:45 AM
smart_toy

当然。自定义 Hook 是一个函数，其名称以 "use" 开头，它允许你将组件逻辑提取到可重用的函数中。

这是一个经典的 useLocalStorage 例子，它可以让你像使用 useState 一样同步地管理本地存储：

function useLocalStorage(key, initialValue) { const [storedValue, setStoredValue] = useState(() => { try { const item = window.localStorage.getItem(key); return item ? JSON.parse(item) : initialValue; } catch (error) { return initialValue; } }); const setValue = value => { setStoredValue(value); window.localStorage.setItem(key, JSON.stringify(value)); }; return [storedValue, setValue]; }

这个 Hook 封装了 localStorage 的存取逻辑，使得多个组件可以轻松共享同一套存储机制。
